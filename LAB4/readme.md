
# JavaScript Quiz Game

A simple quiz built with JavaScript using prompt and alert. The game asks questions, checks answers, and shows the score.

## Features
- Multiple questions
- Case-insensitive answer checking
- Score tracking
- Option to stop the quiz

## How to Run
1. Copy the code into a `.html` file inside `<script>` tags.
2. Open the file in a browser.
3. Answer the questions in the prompts.
4. See your final score at the end.

## Example Questions
- What is the capital of France? → paris  
- How many continents are there on Earth? → 7  
- Which language is primarily used for web development? → js  
- What is 12 - 4? → 8  
- Which tag is used to display images in HTML? → img  

## Scoring
- Correct answer adds 1 point  
- Wrong answer shows the correct answer  
- Final score is displayed as `Your final score: X / Total Questions`

## Customization
- Edit `questions[]` and `answers[]` arrays to change the quiz
- Adjust feedback messages in `alert()` calls
